# randomUserAgent
Powershell script that gets a random user agent from text file.

### How To Use
>ISE
```sh
.\randomUserAgent.ps1
```
>Windows CMD
```sh
powershell -executionpolicy bypass -File .\randomUserAgent.ps1
```
>Linux Terminal
```sh
pwsh randomUserAgent.ps1 -file "C:\Users\Kintoki\user_agents.txt"
```
